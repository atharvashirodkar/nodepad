About Nodepad

Nodepad :

Nodepad, A Notes Taking App is built using pure Node.js and serves as a simple platform to create and manage notes efficiently.

Features

  • Create and manage text notes
  • Notes stored as .txt files
  • Simple and intuitive interface
  • No external dependencies

Technology Stack:

Node.js HTTP Module FS Module HTML5 CSS3
